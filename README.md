## General info

This app is using axios to fetch data from Poke API. It is made using React version 16.13.1.

## Technologies

Project is created with:

- React version: 16.13.1
- React-Router
- SCSS

## Setup

To run this project, install it locally using npm:

```
$ cd ../pokedex
$ npm install
$ npm start
```
